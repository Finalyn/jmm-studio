# JMM — Studio de création de jeux

Site vitrine de **JMM**, studio de création de jeux multi-plateformes.

- **Harmonie** — serveurs FiveM (GTA RP)
- **Boxit** — serveurs Roblox
- **Coming soon** — jeux web, mobile et PC (dont City Guess)

## Stack

Site **statique** (HTML / CSS / JS pur), sans framework. Polices Syne + Inter.

## Structure

- `index.html` — accueil (hero vidéo, derniers jeux, univers, communauté)
- `harmonie.html` · `boxit.html` · `coming-soon.html` — pages dédiées
- `styles.css` — styles
- `videos/` — vidéos de fond
- `images/` — visuels des jeux

## Développement

Ouvrir `index.html` dans un navigateur. Aucune dépendance à installer.
